# TesSaRes
## What is TesSaRes?
TesSaRes is a project that aims to provide a block stacking experience for everyone, whether a beginner, advanced, or grandmaster-level in the genre. 

# Ways to Play
*Note: This game is a work in progress, and as such, the listed modes may not yet be implemented.*

## Classic
*Methods of play you can find just about anywhere else. The classic block stacking experience*
* **40 Lines** -- Clear 40 lines as fast as possible.
* **Blitz** -- Get as many points as possible in 2 minutes.
* **Endless** -- Play TesSaRes for as long as you can survive.

## Skill Analysis
*Methods of play that analyze your skill and assign you a grade.*
* **Easy** -- An easy mode with no time pressure. Get enough lights to pass each section!
* **Spectrum** -- The regular Skill Analysis-style gameplay. Can you become a Grandmaster?
* **3i** -- A version of play with a speed curve that changes depending on your own skill.
* **Maximum** -- A super hard mode. *20G* all the time!
* **And more...**
